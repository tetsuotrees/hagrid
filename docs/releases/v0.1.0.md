# Hagrid v0.1.0

Release date: 2026-03-04

## Summary

Hagrid v0.1.0 introduces the first production-ready local-first secret lifecycle observer for a single machine.

## Highlights

- deterministic secret reference identity (stable across scans)
- keyed HMAC fingerprints (no secret values persisted)
- encrypted SQLCipher index with Keychain-backed master secret
- scan + suggest + group + drift CLI workflow
- JSON outputs for automation on read-only commands

## Included Commands

- `hagrid init`
- `hagrid scan`
- `hagrid status`
- `hagrid list`
- `hagrid show`
- `hagrid suggest`
- `hagrid group`
- `hagrid ungroup`
- `hagrid drift`
- `hagrid forget`
- `hagrid export`

## Security Notes

- no plaintext secret values stored in index
- no network calls by default
- database encrypted at rest with SQLCipher
- macOS Keychain used for master secret storage

## Known Limitations

- macOS-only in v0.1
- no policy engine (`audit`) yet
- no watcher daemon (`watch`) yet
- no interactive rotation workflow yet
